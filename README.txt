Contributors: Neil Haggerty

Program Description: This program allows mass generation of puzzle games. These games are similar to a crossword, or a word find. It supports 6 different types of puzzles. Given a text file full of quotes, the program uses these quotes to generate puzzles of the choosen type. The program then exports these puzzles to a ms powerpoint presentation. The program supports two languages; English, and Telugu. Each puzzle type includes a preferences class to allow the customization of things such as color, font type, number of puzzles, etc...
